Holes on a shape

A parametric 3D cuboid generator built with Three.js that supports dynamic hole creation and real-time geometry updates. The model recalculates vertices instantly as parameters change, demonstrating procedural geometry manipulation, boolean-style modifications, and interactive orbit controls.

Cutting a extruded box

An interactive 3D Extruded shape cutting simulation built with Three.js. The model dynamically applies angular changes to that extruded geometry while validating geometric constraints to prevent invalid configurations. Demonstrates procedural geometry manipulation, vertex-level transformations etc..

Snake Game

The Snake Game is a classic arcade-style game where the player controls a growing snake that moves continuously within a bounded area. The objective is to collect food items that appear randomly on the screen. Each time the snake eats food, it grows longer, increasing the difficulty of navigation.
